# Changelog

所有重要变更记录在此文件中，版本遵循语义版本。

## 0.2.0 - 2026-07-23

### Added

- 节点收藏、收藏筛选与收藏节点路由策略。
- 连接支持自动、固定国家、收藏节点和固定节点四种路由模式，并在自动切换候选选择中严格应用策略。
- 批量 fast/full 扫描改为有界并发后台任务，前端显示实时完成数、成功数、失败数和最终状态。
- 超级管理员可在 Web 中执行无副作用运行诊断，查看数据库、执行器、TUN、OpenVPN、3proxy、防火墙和受限 Root Helper 的就绪状态。
- 日志中心支持导出当前筛选页的结构化脱敏 JSON。
- GitHub 校验引导脚本新增 `--upgrade`，可在不重建管理员的情况下调用原子升级与自动回滚流程。

### Security

- 新增诊断仅执行文件、可执行依赖与数据库可达性检查，不运行特权命令或修改网络。
- 所有新真实网络路径继续受既有显式环境门禁约束；默认仍使用模拟执行器。

### Migration

- Alembic 新增收藏表，以及连接路由模式和首选国家字段；`scripts/upgrade.sh` 会随原子升级自动迁移并在失败时回滚。

## 0.1.4 - 2026-07-23

### Fixed

- 安装完成时主动执行 `reload-or-restart`，确保软件包安装阶段已经启动的 Nginx 立即加载新增的 HTTPS 站点配置。
- Fast 扫描识别 VPNGate API 返回的非限定主机标签，跳过无效的 DNS 解析并直接探测已校验的公网 IP；真正的 FQDN 仍执行 DNS/IP 一致性校验。

## 0.1.3 - 2026-07-22

### Fixed

- 运行期统一通过虚拟环境的 Python 模块入口启动 Uvicorn、Alembic 和 Root Helper，避免执行移动虚拟环境中仍指向暂存目录的控制台脚本。
- Root Helper 改为固定的 root 所有包装器，并使用 Python 隔离模式加载已安装模块；sudoers 权限范围保持不变。

## 0.1.2 - 2026-07-22

### Fixed

- 修复发布暂存目录由 `mktemp` 创建为 `0700` 后直接激活，导致服务账户无法进入 `/opt/vpngate-manager` 并执行虚拟环境的问题。
- 发布树在激活前显式设置顶层目录为 root 所有的 `0755`，应用代码和虚拟环境仍保持不可由服务账户修改。

## 0.1.1 - 2026-07-22

### Fixed

- 修复可选密码或文件不存在时，`set -e` 导致安装、密码重置、修复、升级或卸载脚本静默退出的问题。
- GitHub 一键安装器现在通过终端安全读取并确认管理员密码，非交互环境仍可使用环境变量传入。
- 已校验的 GitHub 发行包默认使用内置前端，避免 Debian 12 自带 Node.js 与新版 pnpm 不兼容。

## 0.1.0 - 2026-07-22

### Added

- FastAPI、SQLAlchemy、Alembic、Cookie/CSRF 认证、角色权限和审计日志。
- Vue 管理台，覆盖仪表盘、节点、连接、解锁检测、日志、用户与设置。
- VPNGate 安全抓取和解析、节点扫描、出口 IP 分类及四类服务检测。
- 模拟优先的 Namespace、OpenVPN、SOCKS5、Kill Switch、连接生命周期和自动切换。
- 固定操作 Root Helper、最小 sudoers 规则以及凭据和日志脱敏。
- 安装、原子升级与回滚、修复、诊断、密码重置和安全卸载脚本。
- 跨模块模拟验收、启动 fail-closed 恢复、可复现发布包和逐文件摘要清单。

### Security

- 默认关闭全部真实网络功能；每一层真实集成都需要显式环境门禁。
- 禁止 `shell=True` 和 `NOPASSWD: ALL`，所有特权参数均由后端生成或严格校验。
- 发布归档排除本地数据库、凭据、环境文件、缓存和构建工作目录。
