# 参考项目功能映射

本版本在产品行为层面参考了 `baoweise-bot/aimili-vpngate` 的节点收藏、路由偏好、维护进度和诊断体验。审阅基准为提交 `fb22c57717307466c7365e8a0e85ff0afa767493`。

参考仓库采用 GPL-3.0。本项目没有复制、移植或派生其 Python、Shell、HTML、CSS 或 JavaScript 源码，也没有将其文件加入发布包；功能均在本项目现有 FastAPI、SQLAlchemy、Alembic、Vue、RBAC、审计、模拟执行器和受限 Root Helper 架构上独立实现。因此本次升级不擅自替项目所有者选择或改变许可证。

## 独立实现的功能

- 节点收藏与收藏筛选。
- 自动、固定国家、收藏节点、固定节点四种连接路由策略。
- 可查询进度的有界并发批量扫描任务。
- 无副作用的运行依赖与真实功能门禁诊断。
- 当前筛选页的结构化脱敏日志导出。

## 保留的安全边界

- 默认模拟执行器；真实 OpenVPN、Namespace、防火墙、SOCKS5、扫描、连接和自动切换仍需逐层显式环境门禁。
- 不修改宿主机默认路由，不允许 `shell=True`，不配置 `NOPASSWD: ALL`。
- 诊断接口不调用特权 Helper，不执行网络命令，不显示密码、Token、Cookie、API Key 或配置正文。
- 发布包通过外部 SHA-256 与内置逐文件清单双重验证。
